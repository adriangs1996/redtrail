The `TerminalFrame` component renders code in a terminal-like UI with traffic light dots and Catppuccin theming.

## Basic Usage

<TerminalFrame title="bash">
```
$ redtrail scan --target 192.168.1.0/24
[*] Starting network scan...
[+] Host 192.168.1.1 is up (0.0023s latency)
[+] Host 192.168.1.42 is up (0.0051s latency)
[*] Scan complete: 2 hosts up
```
</TerminalFrame>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | `'Terminal'` | Window title displayed in the title bar |
| `prompt` | `string` | `'$ '` | Prompt string (reserved for future use) |
