Las instrucciones para instalar Redtrail se añadirán aquí.

## Requisitos previos

- Toolchain de Rust (para compilar desde fuente)
- Una API key de un proveedor LLM (Anthropic u Ollama)

## Instalar desde fuente

```bash
cargo install --path .
```

## Verificar instalación

```bash
rt --version
```
