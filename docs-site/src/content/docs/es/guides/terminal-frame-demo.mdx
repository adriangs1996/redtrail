El componente `TerminalFrame` renderiza código en una interfaz tipo terminal con puntos de semáforo y tema Catppuccin.

## Uso Básico

<TerminalFrame title="bash">
```
$ redtrail scan --target 192.168.1.0/24
[*] Iniciando escaneo de red...
[+] Host 192.168.1.1 activo (0.0023s latencia)
[+] Host 192.168.1.42 activo (0.0051s latencia)
[*] Escaneo completo: 2 hosts activos
```
</TerminalFrame>

## Props

| Prop | Tipo | Default | Descripción |
|------|------|---------|-------------|
| `title` | `string` | `'Terminal'` | Título de la ventana en la barra superior |
| `prompt` | `string` | `'$ '` | Cadena de prompt (reservado para uso futuro) |
